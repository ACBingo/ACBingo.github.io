hexo无法本地访问

  • hexo

今天早晨来了后，打算deploy下昨天写的文章，结果发现hexo server后，本地打不开网页了。经排查，不是浏览器自身的问题。

难道端口被占用了？

git bush执行hexo s没有任何异常啊，但是127.1就是打不开。
关闭hexo s命令并执行netstat -ano。
发现127.0.0.1:4000端口果然被占用了。

在博客文件夹server模块下找到hexo\node_modules\hexo-server\index.js，然后修改里面的port。比如我从4000改到了5000。保存

然后执行hexo s打开浏览器
127.0.0.1:5000

nice!

经百度后发现大家也遇到过同样的问题。而且他查出来了端口是被一个叫做FoxitPortect.exe的进程给占用了。
突然想起昨天因为用到pdf所以装了个福昕阅读器。
Fuck！
浪费了我一上午的时间。
果断卸载！
